main page - api reference

this component allows you to connect cameras via an antenna
press E to set the channel. to work, make sure that the antenna frequencies and tunnel channel match
make sure that the camera on the sending side is connected to the tunnel and the tunnel to the antenna
make sure that the antenna on the receiving side is connected to the tunnel and the tunnel is connected to the computer
this component completely borrows its API from the camera so that programs working with the camera can use this component
despite the API being identical to the camera, this component has several methods of its own
component name - camera

methods:

  • isCameraAvailable():boolean - returns true if there is a connection to the camera
  • setCameraChannel(number:1-6) - sets the camera channel from 1 to 6
  • getCameraChannel():number - returns the set channel
